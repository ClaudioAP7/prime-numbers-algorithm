# prime-numbers-algorithm
basic prime numbers algorithm using NodeJS + Express

# Download - Clone:
git clone https://github.com/ClaudioAP7/prime-numbers-algorithm.git

# Installation:
npm install

# To Build Server:
npm run build

# To Start Server:
npm start

# To Start Dev-Server:
npm run dev

# To Visit local Server:
http://localhost:3000/primes/15

# Docker pull image
docker pull claudios7/primenumbers

# Docker run image
docker run -it -p 3000:3000 primenumbers
